# Coq Verification Exercise: Sum of the Greatest List Elements

This repository contains specifications for a function that returns the sum of two greatest elements in a list, and also includes the formal Coq implementation, a correctness proof, and an analysis of its completeness, generalization, and OCaml extraction challenges.
